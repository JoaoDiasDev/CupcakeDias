# CupcakeDias - Sistema de Pedidos de Cupcakes
## Visão Geral:

CupcakeDias é um sistema de pedidos de cupcakes online desenvolvido para proporcionar uma experiência conveniente e segura para os clientes realizarem pedidos de cupcakes personalizados, acompanharem o histórico de compras e receberem notificações sobre novidades e promoções. Além disso, o sistema oferece ferramentas de gerenciamento para os funcionários da loja, incluindo controle de estoque, relatórios de vendas e gerenciamento de promoções.
Funcionalidades Principais:

    Cadastro e personalização de novos sabores de cupcakes.
    Realização de pedidos personalizados com opções de escolha de sabores, coberturas e decorações.
    Visualização de relatórios de vendas mensais para acompanhamento do desempenho financeiro da loja.
    Gerenciamento de estoque de ingredientes para garantir disponibilidade de produtos.
    Cálculo automático do frete com base no peso total do pedido e endereço de entrega.
    Gerenciamento de promoções e descontos para incentivar as vendas.
    Recuperação de senha esquecida para facilitar o acesso dos clientes.
    Manutenção de histórico de pedidos para consulta e repetição de pedidos anteriores.
    Notificação aos clientes sobre novidades, promoções e eventos especiais.
    Visualização detalhada de produtos disponíveis com informações sobre ingredientes, preço e avaliações.
